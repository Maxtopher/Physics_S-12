Assignment 7: Electronic Input Devices

Use and create your own sensor. Calibrate the sensor, with its min, max and whether you can control this range.

The Hall Sensor

I had alrady played around with the phototransitor during class time, so I tried to experiment with the Hall sensor for my homework. I wired up the fairly simple circuit and used an extremely basic analogRead function to see the sensor values. With the magnets oriented in one direction, the sensor read values from 10 to about 520. Oriented in the other direction, the sensor read values from 520 to 1014. Therefore, the sensor read about 520 when experiencing no magnetic field. The only way I could find to constrain the range was to limit the distance from the Hall Sensor the magnet was allowed to be. Therefore, the only way to control this range would be to physically limit the freedom of the magnet and its corresponding magnetic field.

Creating a Sensor with Capacitance

Capacitor Sensor

To create my own sensor, I cut out and attached tinfoil pieces to the end of a wire. I used the following circuit configuration and code to measure the output value of my capacitor. The sensing function actually measures the time it takes the plates to charge in arbitrary units, however, this corresponds directly to capacitance. The closer the two plates are to each other, the greater the capacitance and time to charge. The circuit I created also contains two LEDs. The red light is illuminated when there is capacitance readings under 1000, and begins to dim once it reaches 1000. From here, the blue light increaes in brightness with greater capacitance readings. The minimum reading I could get without getting seemingly random low numbers was about 100, while the greatest value I recorded was 92340, with just a piece of tape between the larger tin foil piece and the grounded sensor. Therefore, I found the the minimum value is fairly constant, however, the upper bound can be altered by varying the size of the capacitor area. With smaller capacitor pieces, there is a dramatically smaller upper bound on the range of capacitance measured.
