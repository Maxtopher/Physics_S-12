Assignment 3: Tools & Electronics

Make a circuit using the breadboard and kit components. Then use the multimeter to measures voltages and calculate currents. Simulate the circuit, and then use the Microcontroller Analog to measure voltage of a potentiometer or voltage divider.

Making My First Circuit

Basic LED Light Circuit

I haven't had any experience creating circuits before this class. Therefore, starting with the basic LED light circuit was a perfect way to start. I stripped the wires trying to leave just enough exposed wire to be entired contained in the bread board. Next, I gathered the necessary components: the microcontroller, breadboard, LED, 100 Ohm resistor, wires and multimeter. Becasue this was my first circuit, I decided to first model the circuit using the online circuit simulator tool, Tinkercad. With a functional circuit modeled, I started to create my first physical circuit. Attaching the red wires to the 3.3V power source and the black wires to the ground, I simply had to complete the circuit with a resistor and the LED light (making sure to put the cathode towards ground). With this simple circuit, the resistor and LED were in series, so there were two possible cofigurations for the circuit. I tried them both below:

Configuration 1

The first configuration I created and measured was the LED follwed by the resistor. Using the multimeter, I calculated the voltage drop over each component of the circuit. The LED had a voltage drop of 2.81 Volts and the resistor had a voltage drop of 0.49 Volts. Calculating the current of this circuit over the resistor, we have I=4.9x10-3 A.

Configuration 2

For the second configuration, the resistor is followed by the LED. Despite the changing of orientation, each component is defined by the same voltage drop as in the original configuration. This means that the current in both circuits are equivalent. This makes sense as the total impedance to current would be equivalent in both circuits.

The Potentiometer

Potentiometer and the Multimeter

The first circuit I created using the potentiometer, I utilized an open yellow wire from the cenrtal pin of the potentiometer. The multimeter was used to complete this circuit and measure the variable voltage with respect to the ground initiated by the potentiometer. At zero resistance, we read the full voltage potential of 3.3V, while at the maximum resistance, we read that the voltage drop to the ground is 0V. Therefore, the potentiometer could be "tuned" to allow any level of voltage (from 0-3.3V) through the central pin.

Analog Read Serial

After learning how to upload code from my computer to the microcontroller using the basic "Blink" code, I wanted to adapt my circuit to measure the voltage across the cenrtal pin of the potentiometer on my computer. To do this, I simply moved the open yellow wire into the A0 input port on the microcontroller and ran the "AnalogReadSeries" code. Now, in the serial monitor view, I could see the real time voltage across the central pin as the dial of the potentiometer was turned. This was represented as a number ranging from 0 to 1023. Essentially, the voltage was equivalent to the starting voltage times the output over 1023. Thus, if the serial monitor read a 512, this was equivalent to 50% of the total voltage, or 1.65V. Using Tinkercad, I was able to simulate both versions of this circuit (measured using the multimeter and the analog measurement code). I also attempted to slightly change the basic "AnalogReadSeries" code by getting it to convert the initial output automatically into the voltage. I had limited success with this, and was only able to get the program to round to an integer of the voltage.

Voltage Divider

Two Equal Resistors

For my first attempt at creating a voltage divider, I used two equivalent resistors (100 Ohms). Becasue these resistors were equivalent, the circuit should have equally dissipated the voltage across each resistor (1.65V each). This was predicted by the formula for resistors in series and the simulation of the circuit. Ultimately, this prediction was verified using the multimeter. However, I wanted to use the "AnalogReadSeries" code in place of the multimeter as we did previously with the potentiometer. Using the end of the yellow wire not in the A0 input, I was able to use this exposed wire similarly to a multimeter. Where it touched, the serial monitor read the voltage with respect to ground.

Two Different Resistors

In the next circuit, I experimented with two different resistors. The first resistor was 100 Ohms while the second was 470 Ohms. For two resistors in parallel, the voltage across the first is equivalent to the total voltage times the first resistor over the sum of both resistors. Both the calculations and simulation predicted a voltage drop of 0.58 Volts across the 100 Ohm resistor and a voltage drop of 2.72 Volts across the 470 Ohm resistor. This was quickly verified experimentally using the newly created circuit and the multimeter to physically measure these voltages.

Thermistor

Using Voltage Divider with Thermistor

To experiment with the thermistor, I first had to find it's range of resistance. At room temperature, the thermistor had a resistance of 10KOhms, but when heated, the resistance decreased to around 6KOhms. Therefore, I used a 10KOhm resistor to create my voltage divider. I first tested my circuit using two resistors of 10KOhms. To verfiy that it split the voltage evenly, I again attempted to convert the analog sensor reading from the unitless designation of 0 to 1023 into an actual voltage. This time I inserted a formula of "sensorValue*3.3/1023". This printed the correct voltage without having to manually convert the numbers. With this working properly, i created my circuit for the thermistor. Inserting the voltage reader between the thermistor and the resistor, I was able to measure how the voltage across the thermistor changed with different temperatures. At room themperature, the thermistor held steadily around 1.77V, but when heated with my fingers, it increased to about 1.95V. This makes sense, becasue with less resistance within the thermistor at higher temperatures, it should allow more voltage to pass through unimpeded. Ultimately, an increase in temperature results in an increase in voltage across the thermistor which can be measured using the analog input function of the microcontroller.
