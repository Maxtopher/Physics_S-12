Introduction to Digital Fabrication

Physics S-12, Harvard Summer School, 2021

2021 Phys S-12 Website

Weekly Assignments:

Assignment 1:

Introduction & Documentation

Creating a website with an "About Me" page and a "Final Project Proposal". Additionally, reading Chapters 1-3 in Making Things Move and researching local fab labs or makerspaces.

Assignment 2:

2D Design & Cutting

Learning how to use Fusion 360, creating an assembly of multiple components, and modeling components from the kit.

Assignment 3:

Electronics & Tools

Make a circuit using the breadboard and kit components. Then use the multimeter to measures voltages and calculate currents. Simulate the circuit, and then use the Microcontroller Analog to measure voltage of a potentiometer or voltage divider.

Assignment 4:

Rapid Prototyping

Create a kinetic sculpture using an electronic circuit with physical components.

Assignment 5:

Programmable Electronics

Program a microcontroller to produce an output using at leat one button for an input. Include a conditional statement and a for loop.
