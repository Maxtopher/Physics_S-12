Assignment 5: Programmable Electronics

Program a microcontroller to produce an output using at leat one button for an input. Include a conditional statement and a for loop.

Circuitry Practice

Circuitry and coding are still very new to me. To begin this assignment, I wanted to practice a variety of simple coding strategies and circuit designs. This practice began with the simple process of learning about simple pullup, pulldown, and input_pullup circuit designs. With a pulldown resistor in the circuit, a pressed button will report as HIGH. With a pullup resistor, a pressed button would report as LOW. The same is true of the internal (input) pullup resistor in the microcontroller. Shown are these the types.

Simple Buttons and LED

Turning an LED On and Counting Presses

Continuing my circuitry practice, I tried to transition to controlling an LED on my breadboard rather than the builtin LED on the microcontroller. This was a fairly simple switch, so I experimented with multiple configurations. I also tried to put multiple buttons in parallel to make the light only turn on when each button was pressed at the same time.

To continue the challenge, I created a code that would also count the number of times the light was turned on. The code for the counter and Button controlled LED can be found here: LED/Button Counter.

2 Buttons, 1 LED

Putting multiple buttons in parallel to control on LED was a simple way to require multiple buttons to be pressed in order to turn on an LED. However, wiring two buttons to have two separate inputs made the coding a little more challenging. Here, it was key to use a conditional and the && logical operator. This code requires both buttons to be pressed in order to turn the LED on. 

if (button1State == LOW && button2State == LOW){
    analogWrite(LED, 150);
  } else {
    analogWrite(LED, 0);
  }
The complete code can be found here: 2 Buttons, 1 LED Code.

Buttons and LEDs on One Circuit

Next I tried to add multiple compoents to the same circuit and code. Becasue I now knew how to create button/LED code, I tried to create a circuit with two buttons that individually controlled two separate LEDs. This circuitry and code was very similar to the previous example, except there was no counter and each button had it's own conditional statement defined. The following was the loop() code for this design: 

void loop() {
  buttonState1 = digitalRead(button1);
  buttonState2 = digitalRead(button2);
  
  if (buttonState1 == LOW) { 
    digitalWrite(ledPin1, HIGH);
  } else {
    digitalWrite(ledPin1, LOW);
  }

  if (buttonState2 == LOW) {
    digitalWrite(ledPin2, HIGH);
  } else {
    digitalWrite(ledPin2, LOW);
  }
This allows each LED to be controlled individually by a different button. The complete code can be found here: MultiLED Code.

Blinking Potentiometer

A potentiometer can be used to alter the speed at which an LED blinks on and off. The key to this code is defining a variable for the potentiometer "Sensor Value". Simply by using the code delay(sensorValue), after programming the LED to turn on, and again after programming it to turn off, the rate at which the LED blinks is directly controlled by the input from the reading from the potentiometer. The complete code can be found here: Blinking Potentiometer Code.

2 LED for() Loop

Wiring two LEDs separately with input_pullup, we can use a nested for() loop in order to have each LED blink at different rates. The nested for() loop in this code requires the light to blink on and off each second for a minute before the outside for() loop and be completed and the second light turns on. Therfore, this code has one LED blinking at one second intervals while the other LED blinks at one minute intervals. The complete code can be found here: LED for() Loop Code.

LED Button Dimmer

My last practice circuit and code was creating a button that would light up an LED, and then the LED would dim until it was off in about 2 seconds. To accomplish this, another nested for() loop inside a conditional worked best. This specifies for the integer I to quickly dim from 255 to 0, at intervals of 1 very quickly. Therfore, when the button is pressed, the LED is illuminated and then quickly dims until off. 

void loop() {
  if (digitalRead(buttonPin) == LOW) {
    for(int i = 255; i >= 0; i--) {
    analogWrite(ledPin, i);
    delay(10);
    }
  }
The complete code can be found here: Button Dimmer Code.

Final Circuit

My final circuit and code combined the functionality of the potentiometer and the previous LED button dimmer example. I added a second LED to be dimmed and used the potentiometer to control how fast the LEDs were dimmed. To do so, I had to map the input from the potentiometer to a more resonable timeframe for the dimming. Therefore, I used the map command to write sensorValue = map(analogRead(sensorPin), 0, 1023, 1, 15). From here, I used the sensorValue to be the variable within the delay command. The complete code can be found here: Dimmer/Potentiometer Code.
